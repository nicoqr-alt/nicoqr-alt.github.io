Proyecto destacado

Plataforma editorial digital del Instituto de Matemáticas (UNAM)

Python · MkDocs · GitHub · CSV → Páginas estáticas

Desarrollo de una página web responsiva para la difusión de libros y textos del Instituto de Matemáticas. Diseñé el flujo de trabajo y pipelines en Python para generar páginas automáticamente a partir de archivos CSV, reduciendo el trabajo manual y los errores.
